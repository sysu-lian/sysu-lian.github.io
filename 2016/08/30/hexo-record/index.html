快速发布博客

欲只需一句命令，将blog内容，发布于github服务器。
配置 站点配置文件

执行 hexo deploy
出现错误提示：
error deployer not found:git

原因分析：
hexo-deployer-git plugin过期

解决方案：
重新安装该plugin即可
npm install hexo-deployer-git –save
